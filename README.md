# Top 5 Lister

A social platform where users can create, like, dislike, and comment on user-generated top 5 lists.

## Tech Stack
- **Frontend:** React, Material UI
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Auth:** JSON Web Tokens (JWT)

## Features
- Create and share top 5 lists on any topic
- Like, dislike, and comment on other users' lists
- JWT-based user authentication
